# BDpostgreSQLFutebol
Banco de dados relacional PostgreSQL populado com dados sobre Futebol

## Estrutura
Desenvolvemos a camada de Apresentação, que é a interface gráfica e a camada de Persistência que acessa o banco.

![Arquitetura em camadas](http://image.slidesharecdn.com/poo-22-javabancodedados-120712114638-phpapp01/95/poo-21-java-e-banco-de-dados-2-728.jpg?cb=1342094090)

## Executando
Para executar, o projeto deve ser importado no Eclipse e depois é só apertar RUN.
